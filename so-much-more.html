3 So Much More

R language is use in many industries such as research in academia, pharmaceutical for clinical trials, finance for risk management, social media for sentiment analysis, manufacturing for predicting demand and market trends and many more.

Below is an example of a reproducible expression in building a simple metric stock analysis in R combined with shiny.

View Code

library(shiny)
library(plotly)
library(tidyquant)
library(ggplot2)
library(dplyr)
library(shinydashboard)
library(dygraphs)
library(timetk)
library(glue)

# You can change to whatever ticker you want
ticker <- c("NIO", "SPY")
# tidyquant package for tidy format
StockData <- tq_get(ticker, from = "2020-06-01")
# table format
table3.1 <- StockData[1:6,] %>% kable(caption = 'NIO vs. SPY') %>% kable_styling()
